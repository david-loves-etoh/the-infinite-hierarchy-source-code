## 单体宇宙

此世界观最基本的宇宙学结构。
是一片无限大的空间，每个单体宇宙有着不同的自然规律。
对应序数：$\omega$

## 多元宇宙

包含无限个单体宇宙。
对应序数：$\omega^2$

## 高阶多元宇宙

每阶多元宇宙包含无限个低一阶的多元宇宙。
例如，2阶多元宇宙包含无限个多元宇宙，3阶多元宇宙包含无限个2阶多元宇宙，4阶多元宇宙包含无限个3阶多元宇宙……
这种结构无限持续。
对应序数：$\omega^\omega$
